# Car-Share Venture: Financial Model Explorer

An interactive financial modeling tool for analyzing the unit economics of a car-sharing business.

**[Launch the Tool](https://dpdimov.github.io/car-sharing/)**

## Features

- **Pricing Strategy** — Adjust fees and rates, see 5-year projections
- **Cost Structure** — Local office cost breakdown by category
- **Plan vs Actual** — Compare planned vs actual operating data
- **Unit Economics** — View margins per use, per member, or per car
- **Sensitivity Analysis** — Tornado charts and heatmaps
- **Lifetime Value** — Customer LTV with survival curves

All calculations run client-side with no server dependencies.
